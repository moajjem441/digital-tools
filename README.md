# 🚀 Digital Tools

A modern collection of useful digital tools built with **Next.js + React**.  
This project is designed to practice frontend development while building real-world UI features like cart system, product listing, and state management.

---

## 📌 Features

- 🛍️ Product listing system
- 🛒 Add to Cart functionality
- 🔢 Cart counter system
- 💰 Total price calculation
- ⚡ Dynamic state management with React hooks
- 🎯 Clean and responsive UI
- 🔄 Toggle between Products and Cart view

---

## 🧠 What I Learned

- React `useState` and component state flow
- Props drilling and state lifting
- Conditional rendering
- Array methods like `map`, `reduce`, `some`
- UI state management in real apps

---

## 🛠️ Tech Stack

- React.js
- Next.js

- Tailwind CSS

## ⚙️ How to Run Locally

```bash
# Clone the repository
git clone https://github.com/moajjem441/digital-tools.git

# Go to project folder
cd digital-tools

# Install dependencies
npm install

# Run the project
npm run dev


Then open:

http://localhost:3000
