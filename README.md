# 🚀 Digital Tools

A modern collection of useful digital tools built with **Next.js + React**.  
This project is designed to practice frontend development while building real-world UI features like cart system, product listing, and state management.

---

## 🔗 Live Demo
[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen)](https://digital-tools-orcin.vercel.app)




---

## 📌 Features

- 🛍️ Product listing system
- 🛒 Add to Cart functionality
- 🔢 Cart counter system
- 💰 Total price calculation
- ⚡ Dynamic state management with React hooks
- 🎯 Clean and responsive UI
- 🔄 Toggle between Products and Cart view

## 📂 Project Structure
src/
├── all-cards/ # Product listing & main card logic
├── banner/ # Hero banner section
├── card/ # Individual product card component
├── card-separator/ # UI separator for cards section
├── cart/ # Cart page & cart UI
├── footer/ # Footer section
├── navbar/ # Navigation bar
├── ready-part/ # Ready section / UI block
├── threeCards/ # Three card showcase section
├── transparent-card/ # Transparent style card component
├── App.jsx # Main app component
├── App.css # App styling
├── index.css # Tailwind + DaisyUI config styles
└── main.jsx # React entry point

## ⚙️ How to Run Locally

```bash
# Clone the repository
git clone https://github.com/moajjem441/digital-tools.git

# Go to project folder
cd digital-tools

# Install dependencies
npm install

# Run the project
npm run dev


Then open:

http://localhost:3000
